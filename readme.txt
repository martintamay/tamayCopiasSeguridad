#Configuraciones
Todos los archivos de configuración se encuentran en la carpeta settings y son los siguientes

#DATABASES
se listan una abajo de la otra

#DIRECTORIES
se listan uno abajo del otro de forma [dirección];[nombre para guardar]
ejemplo c:/ContabilidadArapy;Sistema Contabilidad

#secure_places
Otras direcciones en las cuales guardar una abajo de la otra, ej.:
D:/copSegs

#configuration
tiene los datos para acceder a la bd
